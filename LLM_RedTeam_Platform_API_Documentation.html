üîó Core API Endpoints

1. Assessment Management

POST /api/assessments

Creates a new red team assessment configuration.

Request Body:

{ "name": "GPT-4 Security Assessment", "description": "Comprehensive red team testing", "llm_provider": "openai", "model_name": "gpt-4", "test_categories": ["jailbreak", "bias", "hallucination", "privacy", "manipulation"] }

Response 201:

{ "success": true, "assessment": { "id": 125, "name": "GPT-4 Security Assessment", "status": "pending", "total_prompts": 125, "created_at": "2024-08-24T10:30:00Z", "llm_provider": "openai", "model_name": "gpt-4", "test_categories": ["jailbreak", "bias", "hallucination", "privacy", "manipulation"] } }

Error Response 400:

{ "error": "Unsupported LLM provider: invalid_provider" }
GET /api/assessments

Retrieves paginated list of all assessments.

Query Parameters:

Parameter Type Default Description
page integer 1 Page number
per_page integer 10 Items per page (max: 100)

Response 200:

{ "assessments": [ { "id": 125, "name": "GPT-4 Security Assessment", "status": "completed", "llm_provider": "openai", "model_name": "gpt-4", "overall_score": 7.2, "safeguard_success_rate": 78.4, "created_at": "2024-08-24T10:30:00Z", "completed_at": "2024-08-24T11:15:30Z" } ], "total": 50, "pages": 5, "current_page": 1 }
POST /api/assessments/{assessment_id}/run

Initiates execution of a pending assessment.

Request Body:

{ "api_key": "sk-your-openai-api-key-here" }

Response 200:

{ "success": true, "message": "Assessment started", "assessment": { "id": 125, "status": "running", "started_at": "2024-08-24T11:00:00Z" } }

3. Connection Testing

POST /api/test-connection

Tests LLM API connection with provided credentials.

Request Body:

{ "provider": "openai", "apiKey": "sk-your-openai-api-key", "model": "gpt-4" }

Success Response 200:

{ "success": true, "provider": "openai", "model": "gpt-4", "response_time": 0.85, "response_preview": "Hello! How can I assist you today?", "message": "OpenAI connection successful" }

Error Responses:

401 Invalid API Key:
{"success": false, "error": "Invalid API key"}
402 Quota Exceeded:
{"success": false, "error": "API quota exceeded"}
429 Rate Limited:
{"success": false, "error": "Rate limit exceeded"}
